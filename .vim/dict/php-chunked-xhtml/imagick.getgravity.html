Imagick::getGravity

(PECL imagick 2, PECL imagick 3)

Imagick::getGravity — gravity を取得する

説明

public Imagick::getGravity(): int

Imagick オブジェクトのグローバル gravity プロパティを取得します。 このメソッドは、ImageMagick バージョン 6.4.0 以降で Imagick をコンパイルした場合に使用可能です。

パラメータ

この関数にはパラメータはありません。

返り値

gravity プロパティを返します。 gravity 定数 の一覧を参照ください。
