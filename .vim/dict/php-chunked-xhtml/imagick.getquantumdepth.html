Imagick::getQuantumDepth

(PECL imagick 2, PECL imagick 3)

Imagick::getQuantumDepth — quantum depth を取得する

説明

public static Imagick::getQuantumDepth(): array

Imagick quantum depth を返します。

パラメータ

この関数にはパラメータはありません。

返り値

"quantumDepthLong" と "quantumDepthString" をキーとして持つ配列を返します。

エラー / 例外

エラー時に ImagickException をスローします。
