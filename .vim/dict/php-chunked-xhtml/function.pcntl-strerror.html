pcntl_strerror

(PHP 5 >= 5.3.4, PHP 7, PHP 8)

pcntl_strerror — 指定した errno に対応するシステムのエラーメッセージを取得する

説明

pcntl_strerror(int $errno): string|false

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

errno

返り値

エラーの説明を返します。失敗した場合に false を返します。
