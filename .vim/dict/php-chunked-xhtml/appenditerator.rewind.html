AppendIterator::rewind

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

AppendIterator::rewind — イテレータを巻き戻す

説明

public AppendIterator::rewind(): void

最初の内部イテレータの最初の要素まで巻き戻します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
