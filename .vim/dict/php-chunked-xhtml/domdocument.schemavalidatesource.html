DOMDocument::schemaValidateSource

(PHP 5, PHP 7, PHP 8)

DOMDocument::schemaValidateSource — スキーマに基づいてドキュメントを検証する

説明

public DOMDocument::schemaValidateSource(string $source, int $flags = 0): bool

指定した文字列に定義されているスキーマに基づいてドキュメントを検証します。

パラメータ

source

スキーマを含む文字列。

flags

Libxml のスキーマ検証フラグのビットマスク。現在サポートしている値は LIBXML_SCHEMA_CREATE だけです。 Libxml 2.6.14 以降で使えます。

返り値

成功した場合に true を、失敗した場合に false を返します。
