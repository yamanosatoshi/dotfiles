shm_get_var

(PHP 4, PHP 5, PHP 7, PHP 8)

shm_get_var — 共有メモリから変数を返す

説明

shm_get_var(SysvSharedMemory $shm, int $key): mixed

shm_get_var() は、 shm で指定した共有メモリセグメントから 変数 key を読みこみます。 読み込んだ変数は、まだ共有メモリに存在します。

パラメータ

shm

共有メモリセグメント。shm_attach() から取得します。

key

変数のキー。

返り値

指定したキーの変数を返します。

変更履歴

バージョン 説明
8.0.0 引数 shm は、 SysvSharedMemory クラスのインスタンスを期待するようになりました。 これより前のバージョンでは、リソースが期待されていました。
