DOMElement::setAttribute

(PHP 5, PHP 7, PHP 8)

DOMElement::setAttribute — 新しい属性を追加する

説明

public DOMElement::setAttribute(string $qualifiedName, string $value): DOMAttr|bool

qualifiedName という名前の属性を、指定した値に設定します。 属性が存在しない場合は、作成されます。

パラメータ

qualifiedName

属性の名前。

value

属性の値。

返り値

新しい DOMAttr、 あるいはエラーが発生した場合は false を返します。

エラー / 例外

DOM_NO_MODIFICATION_ALLOWED_ERR

ノードが読み込み専用の場合に発生します。

例

例1 属性を設定する

<?php
$doc = new DOMDocument("1.0");
$node = $doc->createElement("para");
$newnode = $doc->appendChild($node);
$newnode->setAttribute("align", "left");
?>
