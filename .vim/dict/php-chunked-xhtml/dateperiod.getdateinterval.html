DatePeriod::getDateInterval

(PHP 5 >= 5.6.5, PHP 7, PHP 8)

DatePeriod::getDateInterval — 間隔を取得する

説明

オブジェクト指向型

public DatePeriod::getDateInterval(): DateInterval

期間に適用される間隔を表す DateInterval object を取得する。

パラメータ

この関数にはパラメータはありません。

返り値

DateInterval object を返す。

例

例1 DatePeriod::getDateInterval() の例

<?php
$period = new DatePeriod('R7/2016-05-16T00:00:00Z/P1D');
$interval = $period->getDateInterval();
echo $interval->format('%d day');
?>

上の例の出力は以下となります。

1 day
