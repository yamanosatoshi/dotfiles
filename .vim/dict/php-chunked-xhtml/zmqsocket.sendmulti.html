ZMQSocket::sendmulti

(PECL zmq >= 0.8.0)

ZMQSocket::sendmulti — マルチパートメッセージを送る

説明

public ZMQSocket::sendmulti(array $message, int $mode = 0): ZMQSocket

ソケットを使ってマルチパートメッセージを送ります。 この操作は ZMQ::MODE_NOBLOCK を使わないとブロックされる可能性があります。

パラメータ

message

送信するメッセージ - 文字列の配列

mode

マルチパートメッセージを受け取るか、 ノンブロッキング操作を行うかのモードフラグ。 ZMQ::MODE_* 定数を参照してください。

返り値

現在のオブジェクトを返します。 エラーの場合は ZMQSocketException をスローします。 Returns the current object. Throws ZMQSocketException on error. ZMQ::MODE_NOBLOCK が使われていた場合、 操作はブロックされ、bool false が返されます。
