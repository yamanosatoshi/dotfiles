本章では、PHP を macOS にインストールする際の注意事項とコツを説明します。 PHP は Mac OS X バージョン 10.0.0 以降、 macOS Monterey (12.0.0) より前の MacOS に標準添付されています。 また、コンパイルの手順は Unix へのインストールガイド と同じです。
