unixtojd

(PHP 4, PHP 5, PHP 7, PHP 8)

unixtojd — Unix タイムスタンプをユリウス歴に変換する

説明

unixtojd(int|null $timestamp = null): int|false

Unix タイムスタンプ(1970/1/1 からの秒数)timestamp をユリウス歴に変換して返します。timestamp が 指定されない場合は現在の日付が使用されます。 どちらの場合でも、時刻は地方時間と見なされます(UTCではありません)。

パラメータ

timestamp

変換したい Unix タイムスタンプ。

返り値

ユリウス積算日を表す整数値を返します。 失敗した場合に false を返します

変更履歴

バージョン 説明
8.0.0 timestamp は、nullable になりました。
