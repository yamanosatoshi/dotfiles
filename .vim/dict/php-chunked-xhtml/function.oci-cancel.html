oci_cancel

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

oci_cancel — カーソルからの読み込みをキャンセルする

説明

oci_cancel(resource $statement): bool

カーソルを無効にして関連付けられた全てのリソースを開放し、 読み込みをキャンセルします。

パラメータ

statement

OCI ステートメント

返り値

成功した場合に true を、失敗した場合に false を返します。
