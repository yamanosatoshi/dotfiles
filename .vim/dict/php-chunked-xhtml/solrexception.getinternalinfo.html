SolrException::getInternalInfo

(PECL solr >= 0.9.2)

SolrException::getInternalInfo — 例外がスローされた際の内部情報を返す

説明

public SolrException::getInternalInfo(): array

例外がスローされた際の内部情報を返します。

パラメータ

この関数にはパラメータはありません。

返り値

エラーがスローされた際の内部情報を含む配列を返します。エクステンション開発者によるデバッグのためだけに使われます。
