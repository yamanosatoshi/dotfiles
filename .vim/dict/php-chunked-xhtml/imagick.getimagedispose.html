Imagick::getImageDispose

(PECL imagick 2, PECL imagick 3)

Imagick::getImageDispose — 画像の配置方法を取得する

説明

public Imagick::getImageDispose(): int

画像の配置方法を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に配置方法を返します。

エラー / 例外

エラー時に ImagickException をスローします。
