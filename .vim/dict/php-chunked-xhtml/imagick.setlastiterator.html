Imagick::setLastIterator

(PECL imagick 2 >= 2.0.1, PECL imagick 3)

Imagick::setLastIterator — Imagick イテレータを最後の画像に設定する

説明

public Imagick::setLastIterator(): bool

Imagick イテレータを最後の画像に設定します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に true を返します。
