Iterator::key

(PHP 5, PHP 7, PHP 8)

Iterator::key — 現在の要素のキーを返す

説明

abstract public Iterator::key(): mixed

現在の要素のキーを返します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合にスカラー型、失敗した場合に null を返します。

エラー / 例外

失敗した場合に E_NOTICE を発行します。
