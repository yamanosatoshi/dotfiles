GmagickDraw::getstrokeopacity

(PECL gmagick >= Unknown)

GmagickDraw::getstrokeopacity — オブジェクトの輪郭の不透明度を返す

説明

public GmagickDraw::getstrokeopacity(): float

オブジェクトの輪郭の不透明度を返します。

パラメータ

この関数にはパラメータはありません。

返り値

不透明度を表す浮動小数点数値を返します。
