Imagick::getColorspace

(PECL imagick 3)

Imagick::getColorspace — 色空間を取得する

説明

public Imagick::getColorspace(): int

グローバル色空間の値を取得します。 このメソッドは、ImageMagick バージョン 6.5.7 以降で Imagick をコンパイルした場合に使用可能です。

パラメータ

この関数にはパラメータはありません。

返り値

整数値を返します。この値を COLORSPACE 定数 と比較することができます。
