GmagickDraw::getstrokewidth

(PECL gmagick >= Unknown)

GmagickDraw::getstrokewidth — オブジェクトの輪郭の描画に使う線の幅を返す

説明

public GmagickDraw::getstrokewidth(): float

オブジェクトの輪郭の描画に使う線の幅を返します。

パラメータ

この関数にはパラメータはありません。

返り値

線の幅を表す浮動小数点数値を返します。
