OAuth::disableSSLChecks

(PECL OAuth >= 0.99.5)

OAuth::disableSSLChecks — SSL チェックを無効にする

説明

public OAuth::disableSSLChecks(): bool

通常の SSL ピア証明書およびホストのチェックを無効にします。 これは実運用に使うものではありません。 あるいは、sslChecks メンバーに false を設定することによっても SSL チェックを無効にできます。

パラメータ

この関数にはパラメータはありません。

返り値

true

変更履歴

バージョン 説明
PECL oauth 0.99.8 sslChecks メンバーが追加されました。
