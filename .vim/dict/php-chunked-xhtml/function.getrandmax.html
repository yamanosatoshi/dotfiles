getrandmax

(PHP 4, PHP 5, PHP 7, PHP 8)

getrandmax — 乱数の最大値を取得する

説明

getrandmax(): int

rand() をコールすることにより得られる最大の値を返します。

パラメータ

この関数にはパラメータはありません。

返り値

rand() が返す乱数の最大値を返します。
