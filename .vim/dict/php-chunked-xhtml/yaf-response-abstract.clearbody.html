Yaf_Response_Abstract::clearBody

(Yaf >=1.0.0)

Yaf_Response_Abstract::clearBody — 既存のすべてのレスポンス本文を破棄します

説明

public Yaf_Response_Abstract::clearBody(string $key = ?): bool

既存のコンテンツを消去します。

パラメータ

key

コンテンツのキー。キーを指定しない場合は、すべてのコンテンツを消去します。

注意:

このパラメータは 2.2.0 で導入されました。
