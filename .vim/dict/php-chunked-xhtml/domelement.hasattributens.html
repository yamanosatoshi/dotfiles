DOMElement::hasAttributeNS

(PHP 5, PHP 7, PHP 8)

DOMElement::hasAttributeNS — 属性が存在するかどうかを調べる

説明

public DOMElement::hasAttributeNS(string|null $namespace, string $localName): bool

要素のメンバとして localName という名前の属性が 名前空間 namespace に存在するかどうかを示します。

パラメータ

namespace

名前空間 URI。

localName

ローカル名。

返り値

成功した場合に true を、失敗した場合に false を返します。
