Imagick::destroy

(PECL imagick 2, PECL imagick 3)

Imagick::destroy — Imagick オブジェクトを破棄する

説明

public Imagick::destroy(): bool

Imagick オブジェクトを破棄し、関連するリソースをすべて開放します。 このメソッドは非推奨です。かわりに Imagick::clear を使いましょう。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に true を返します。
