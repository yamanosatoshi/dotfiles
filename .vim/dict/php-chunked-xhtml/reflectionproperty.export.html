ReflectionProperty::export

(PHP 5, PHP 7)

ReflectionProperty::export — エクスポートする

警告

この関数は PHP 7.4.0 で 非推奨 になり、PHP 8.0.0 で 削除 されました。この関数に頼らないことを強く推奨します。

説明

public static ReflectionProperty::export(mixed $class, string $name, bool $return = ?): string

エクスポートします。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

argument

エクスポートするリフレクション。

name

プロパティ名。

return

true に設定すると、エクスポートを発行するのではなく返します。 false (デフォルト) の場合はその逆です。
