SolrInputDocument::toArray

(PECL solr >= 0.9.2)

SolrInputDocument::toArray — 入力した文書の配列表現を返す

説明

public SolrInputDocument::toArray(): array

入力した文書の配列表現を返します。

パラメータ

この関数にはパラメータはありません。

返り値

フィールドを含む配列を返します。失敗時には false を返します。
