SolrDocument::reset

(PECL solr >= 0.9.2)

SolrDocument::reset — SolrDocument::clear()へのエイリアス

説明

public SolrDocument::reset(): bool

これは、SolrDocument::clear()へのエイリアスです。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に true を、失敗した場合に false を返します。
