Gmagick::normalizeimage

(PECL gmagick >= Unknown)

Gmagick::normalizeimage — カラー画像のコントラストを強調する

説明

public Gmagick::normalizeimage(int $channel = ?): Gmagick

カラー画像のコントラストを強調します。 ピクセルの色を、とりうる値の範囲全体に拡張します。

パラメータ

channel

どのチャネルを正規化するか。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
