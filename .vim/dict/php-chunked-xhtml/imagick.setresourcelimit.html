Imagick::setResourceLimit

(PECL imagick 2, PECL imagick 3)

Imagick::setResourceLimit — 特定のリソースの制限を設定する

説明

public static Imagick::setResourceLimit(int $type, int $limit): bool

このメソッドを使って、ImageMagick ライブラリのリソース制限を変更します。

パラメータ

type

リソースタイプ定数 のうちのひとつ。 単位は制限されるリソースのタイプに依存します。

limit

リソースの制限。単位は、制限するリソースの種類によって異なります。

返り値

成功した場合に true を返します。
