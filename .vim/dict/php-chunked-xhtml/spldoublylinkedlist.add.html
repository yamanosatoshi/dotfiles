SplDoublyLinkedList::add

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

SplDoublyLinkedList::add — 特定のインデックスに新しい値を 追加/挿入 する

説明

public SplDoublyLinkedList::add(mixed $index, mixed $newval): void

指定された index に newval を挿入し、 以前その位置にあった値(とそれ以後にあるリストの値すべて)をシャッフルします

パラメータ

index

新しい値が挿入されるインデックス

newval

index に入る新しい値

返り値

値を返しません。

エラー / 例外

index が範囲外だったり、 index が数値としてパースできない場合は、 OutOfRangeException がスローされます。
