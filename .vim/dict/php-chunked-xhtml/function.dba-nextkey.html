dba_nextkey

(PHP 4, PHP 5, PHP 7, PHP 8)

dba_nextkey — 次のキーを取得する

説明

dba_nextkey(resource $dba): string|false

dba_nextkey() は、 が指すデータベースの次のキーを返し、内部キーポインタを進めます。

パラメータ

dba

dba_open() あるいは dba_popen() によって返されたデータベースハンドル。

返り値

成功した場合にキー、失敗した場合に false を返します。
