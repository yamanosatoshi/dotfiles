SplFixedArray::valid

(PHP 5 >= 5.3.0, PHP 7)

SplFixedArray::valid — 配列にまだ要素があるかどうかを調べる

説明

public SplFixedArray::valid(): bool

配列にまだ要素があるかどうかを調べます。

パラメータ

この関数にはパラメータはありません。

返り値

配列にまだ要素がある場合に true、それ以外の場合に false を返します。
