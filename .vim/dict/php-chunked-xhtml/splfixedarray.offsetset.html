SplFixedArray::offsetSet

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

SplFixedArray::offsetSet — 指定した添字の新しい値を設定する

説明

public SplFixedArray::offsetSet(int $index, mixed $newval): void

index で指定した添字の値を newval に設定します。

パラメータ

index

設定したい添字。

newval

index の新しい値。

返り値

値を返しません。

エラー / 例外

index が配列で定義されているサイズの範囲外の場合や index を数値として解釈できない場合に RuntimeException をスローします。
