SolrResponse::success

(PECL solr >= 0.9.2)

SolrResponse::success — リクエストが成功したか

説明

public SolrResponse::success(): bool

サーバーへのリクエストが成功したかチェックするために使用されます。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合 true 、失敗した場合 false 。
