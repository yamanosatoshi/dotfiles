定義済み定数

LIBENCHANT_VERSION (string)
libenchant のバージョン。 PHP 8.0.0 以降で利用可能です。
ENCHANT_MYSPELL (int)
MySpell 用の辞書の型。 enchant_broker_get_dict_path() や enchant_broker_set_dict_path() で利用します。 PHP 8.0.0 以降では推奨されません。
ENCHANT_ISPELL (int)
Ispell 用の辞書の型。 enchant_broker_get_dict_path() や enchant_broker_set_dict_path() で利用します。 PHP 8.0.0 以降では推奨されません。
