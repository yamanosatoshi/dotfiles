eio_nready

(PECL eio >= 0.0.1dev)

eio_nready — 未処理のリクエスト数を返す

説明

eio_nready(): int

パラメータ

この関数にはパラメータはありません。

返り値

eio_nready() は、未処理のリクエスト数を返します。
