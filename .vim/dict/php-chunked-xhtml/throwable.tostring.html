Throwable::__toString

(PHP 7, PHP 8)

Throwable::__toString — スローされたオブジェクトの文字列表現を取得する

説明

abstract public Throwable::__toString(): string

パラメータ

この関数にはパラメータはありません。

返り値

スローされたオブジェクトの文字列表現を返します。
