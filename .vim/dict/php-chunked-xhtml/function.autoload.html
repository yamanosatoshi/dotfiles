__autoload

(PHP 5, PHP 7)

__autoload — 未定義のクラスのロードを試みる

警告

この関数は PHP 7.2.0 で 非推奨 になり、PHP 8.0.0 で 削除 されました。この関数に頼らないことを強く推奨します。

説明

__autoload(string $class): void

この関数を定義すれば、クラスのオートローディング を有効にできます。

パラメータ

class

ロードするクラスの名前。

返り値

値を返しません。
