Imagick::valid

(PECL imagick 2, PECL imagick 3)

Imagick::valid — 現在のアイテムが有効かどうかを調べる

説明

public Imagick::valid(): bool

現在のアイテムが有効かどうかを調べます。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に true を返します。
