gmp_div_r

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_div_r — 除算の余りを計算する

説明

gmp_div_r(GMP|int|string $num1, GMP|int|string $num2, int $rounding_mode = GMP_ROUND_ZERO): resource

num1 を num2 で整数として割った際の余りを計算します。余りは、引数 num1 がゼロでない場合に、これと同じ符号を有します。

パラメータ

num1

割られる数。

GMP オブジェクト、整数、あるいは数値に変換可能な数値形式の文字列。

num2

num1 を割る数。

GMP オブジェクト、整数、あるいは数値に変換可能な数値形式の文字列。

rounding_mode

引数 rounding_mode の説明については、関数 gmp_div_q() を参照ください。

返り値

余りを GMP 数で返します。

例

例1 gmp_div_r() の例

<?php
$div = gmp_div_r("105", "20");
echo gmp_strval($div) . "\n";
?>
