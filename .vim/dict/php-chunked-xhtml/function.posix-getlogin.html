posix_getlogin

(PHP 4, PHP 5, PHP 7, PHP 8)

posix_getlogin — ログイン名を返す

説明

posix_getlogin(): string|false

現在のプロセスを所有するユーザーのログイン名を返します。

パラメータ

この関数にはパラメータはありません。

返り値

ユーザーのログイン名を文字列で返します。 失敗した場合に false を返します

例

例1 posix_getlogin() の使用例

<?php
echo posix_getlogin(); //apache
?>
