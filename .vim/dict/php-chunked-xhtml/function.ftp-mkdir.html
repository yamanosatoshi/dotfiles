ftp_mkdir

(PHP 4, PHP 5, PHP 7, PHP 8)

ftp_mkdir — ディレクトリを作成する

説明

ftp_mkdir(resource $ftp, string $directory): string|false

FTP サーバー上に、指定した directory を作成します。

パラメータ

ftp

FTP 接続のリンク ID 。

directory

作成されるディレクトリの名前。

返り値

成功した時には新規に作成したディレクトリ名、エラー時に false を返します。

エラー / 例外

ディレクトリが既に存在していたり、 パーミッションによってディレクトリが作成できなかった場合、 E_WARNING レベルの警告が発生します。

例

例1 ftp_mkdir() の例

<?php

$dir = 'www';

// 接続を確立する
$conn_id = ftp_connect($ftp_server);

// ユーザー名とパスワードでログインする
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// ディレクトリ $dir の作成を試みる
if (ftp_mkdir($conn_id, $dir)) {
 echo "successfully created $dir\n";
} else {
 echo "There was a problem while creating $dir\n";
}

// 接続を閉じる
ftp_close($conn_id);
?>
