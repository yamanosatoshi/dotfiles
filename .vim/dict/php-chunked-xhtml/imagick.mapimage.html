Imagick::mapImage

(PECL imagick 2, PECL imagick 3)

Imagick::mapImage — 画像の色を、参照画像の中の最も近い色に置き換える

警告

この関数は Imagick 3.4.4 以降では 非推奨 になりました。この関数に依存したコードを書くのはやめましょう。

説明

public Imagick::mapImage(Imagick $map, bool $dither): bool

パラメータ

map

dither

返り値

成功した場合に true を返します。

エラー / 例外

エラー時に ImagickException をスローします。
