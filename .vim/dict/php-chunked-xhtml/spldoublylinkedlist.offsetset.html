SplDoublyLinkedList::offsetSet

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

SplDoublyLinkedList::offsetSet — 指定した $index の値を $newval に設定する

説明

public SplDoublyLinkedList::offsetSet(mixed $index, mixed $newval): void

指定した index の値を newval に設定します。

パラメータ

index

設定したいインデックス。

newval

index の新しい値。

返り値

値を返しません。

エラー / 例外

index が範囲外を指すときや index が整数として解釈できないときに OutOfRangeException をスローします。
