APCUIterator::rewind

(PECL apcu >= 5.0.0)

APCUIterator::rewind — イテレータを巻き戻す

説明

public APCUIterator::rewind(): void

イテレータを最初の要素に巻き戻します

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
