is_nan

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

is_nan — 値が非数かどうかを判定する

説明

is_nan(float $num): bool

num が '非数値 (not a number)' であるかどうかを調べます。たとえば acos(1.01) の結果などがこれにあたります。

パラメータ

num

調べる値。

返り値

num が '非数値 (not a number)' の場合に true、そうでない場合に false を返します。

例

例1 is_nan() の例

<?php
// 無効な計算なので、
// NaN 値を返します
$nan = acos(8);

var_dump($nan, is_nan($nan));
?>

上の例の出力は以下となります。

float(NAN)
bool(true)
