DOMElement::hasAttribute

(PHP 5, PHP 7, PHP 8)

DOMElement::hasAttribute — 属性が存在するかどうかを調べる

説明

public DOMElement::hasAttribute(string $qualifiedName): bool

要素のメンバとして qualifiedName という名前の属性が 存在するかどうかを示します。

パラメータ

qualifiedName

要素名。

返り値

成功した場合に true を、失敗した場合に false を返します。
