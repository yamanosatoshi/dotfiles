is_infinite

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

is_infinite — 値が無限大であるかどうかを判定する

説明

is_infinite(float $num): bool

num が（正または負の）無限大である場合に true を返します。たとえば log(0) の結果、 あるいはこのプラットフォーム上で扱える float の範囲を超えた数などが あてはまります。

パラメータ

num

調べる値。

返り値

num が無限大である場合に true、 そうでない場合に false を返します。
