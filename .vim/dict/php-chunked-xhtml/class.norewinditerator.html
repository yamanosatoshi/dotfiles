NoRewindIterator クラス

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

はじめに

このイテレータは巻き戻しの操作を無視します。 これにより、一部のみを走査する 複数の foreach ループでイテレータを処理することが可能になります。

クラス概要
