Imagick::render

(PECL imagick 2, PECL imagick 3)

Imagick::render — それまでのすべての描画コマンドをレンダリングする

説明

public Imagick::render(): bool

それまでのすべての描画コマンドをレンダリングします。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に true を返します。
