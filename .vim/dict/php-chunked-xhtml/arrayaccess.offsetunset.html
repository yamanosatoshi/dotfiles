ArrayAccess::offsetUnset

(PHP 5, PHP 7, PHP 8)

ArrayAccess::offsetUnset — オフセットの設定を解除する

説明

abstract public ArrayAccess::offsetUnset(mixed $offset): void

オフセットの設定を解除します。

注意:

このメソッドは、 (unset) による型キャストの際にはコール されません。

パラメータ

offset

設定解除したいオフセット。

返り値

値を返しません。
