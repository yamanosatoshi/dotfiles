readline_add_history

(PHP 4, PHP 5, PHP 7, PHP 8)

readline_add_history — ヒストリに 1 行追加する

説明

readline_add_history(string $prompt): bool

この関数は、コマンドラインヒストリに 1 行追加します。

パラメータ

prompt

ヒストリに追加する行。

返り値

成功した場合に true を、失敗した場合に false を返します。
