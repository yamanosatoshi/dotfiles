SolrServerException::getInternalInfo

(PECL solr >= 1.1.0, >=2.0.0)

SolrServerException::getInternalInfo — 例外がスローされた際の内部情報を返す

説明

public SolrServerException::getInternalInfo(): array

例外がスローされた際の内部情報を返します。

パラメータ

この関数にはパラメータはありません。

返り値

エラーがスローされた際の内部情報を含む配列を返します。エクステンション開発者によるデバッグのためだけに使われます。
