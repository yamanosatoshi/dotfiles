Imagick::clone

(PECL imagick 2, PECL imagick 3)

Imagick::clone — Imagick オブジェクトの完全なコピーを作成する

説明

public Imagick::clone(): Imagick

Imagick オブジェクトの完全なコピーを作成します。

警告

この関数は imagick 3.1.0 で 非推奨 となりました。 かわりに clone キーワードを使います。

パラメータ

この関数にはパラメータはありません。

返り値

Imagick オブジェクトのコピーを返します。

変更履歴

バージョン 説明
PECL imagick 3.1.0 このメソッドは非推奨となりました。かわりに clone キーワードを使います。

例

例1 Imagick オブジェクトのクローン

<?php
// imagick 2.x および 3.0 の場合
$newImage = $image->clone();

// 3.1.0 以降の場合
$newImage = clone $image;
?>
