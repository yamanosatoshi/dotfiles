OAuth::getCAPath

(PECL OAuth >= 0.99.8)

OAuth::getCAPath — CA 情報を取得する

説明

public OAuth::getCAPath(): array

認証局の情報を取得します。ここには、 OAuth::setCaPath() で設定した ca_path や ca_info が含まれます。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

認証局の情報を含む配列を返します。返される連想配列のキーには ca_path と ca_info が含まれます。

参考

  • OAuth::setCAPath() - CA パスおよび情報を設定する
  • OAuth::getLastResponseInfo() - 直近のレスポンスの HTTP 情報を取得する
