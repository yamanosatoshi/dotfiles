ps_scale

(PECL ps >= 1.1.0)

ps_scale — 縮尺を設定する

説明

ps_scale(resource $psdoc, float $x, float $y): bool

座標系の、水平方向・垂直方向の縮尺をそれぞれ設定します。

パラメータ

psdoc

ps_new() が返す、postscript ファイルのリソース ID。

x

水平方向の倍率。

y

垂直方向の倍率。

返り値

成功した場合に true を、失敗した場合に false を返します。
