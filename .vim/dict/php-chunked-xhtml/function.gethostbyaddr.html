gethostbyaddr

(PHP 4, PHP 5, PHP 7, PHP 8)

gethostbyaddr — 指定した IP アドレスに対応するインターネットホスト名を取得する

説明

gethostbyaddr(string $ip): string|false

ip で指定したインターネットホストのホスト名を返します。

パラメータ

ip

ホストの IP アドレス。

返り値

成功した場合はホスト名を返します。 失敗した場合は、そのままの形の ip を文字列で返します。 入力が壊れている場合は false を返します。

例

例1 単純な gethostbyaddr() の例

<?php
$hostname = gethostbyaddr($_SERVER['REMOTE_ADDR']);

echo $hostname;
?>
