Gmagick::getpackagename

(PECL gmagick >= Unknown)

Gmagick::getpackagename — GraphicsMagick パッケージ名を返す

説明

public Gmagick::getpackagename(): string

GraphicsMagick パッケージ名を返します。

パラメータ

この関数にはパラメータはありません。

返り値

GraphicsMagick パッケージ名を文字列で返します。

エラー / 例外

エラー時に GmagickException をスローします。
