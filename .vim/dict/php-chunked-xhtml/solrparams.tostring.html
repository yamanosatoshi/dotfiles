SolrParams::toString

(PECL solr >= 0.9.2)

SolrParams::toString — オブジェクト内の名前と値のペアのパラメータを全て返します

説明

final public SolrParams::toString(bool $url_encode = false): string

オブジェクト内の名前と値のペアのパラメータを全て返します

パラメータ

url_encode

URL エンコードされた値が返されるかどうか

返り値

成功時に文字列、失敗時に false を返します。
