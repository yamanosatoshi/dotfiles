SolrResponse::getRawResponse

(PECL solr >= 0.9.2)

SolrResponse::getRawResponse — Solr サーバーからの生のレスポンスを返す

説明

public SolrResponse::getRawResponse(): string

Solr サーバーからの生のレスポンスを返します。

パラメータ

この関数にはパラメータはありません。

返り値

Solr サーバーからの生のレスポンスを返します。
