usleep

(PHP 4, PHP 5, PHP 7, PHP 8)

usleep — マイクロ秒単位で実行を遅延する

説明

usleep(int $microseconds): void

指定したマイクロ秒数だけプログラムの実行を遅延させます。

パラメータ

microseconds

実行を停止するマイクロ秒数。マイクロ秒とは、一秒の百万分の一です。

注意: 1000000 より大きな値を指定すること (つまり、1秒以上スリープさせること) は、 オペレーティングシステムによってはサポートしていないかもしれません。 その場合は、sleep() を代わりに使って下さい。

注意: Windows では、与えられたマイクロ秒以上に長くスリープする可能性があります。 動作はハードウェア依存です。

返り値

値を返しません。

例

例1 usleep() の例

<?php

// 現在の時刻
echo date('h:i:s') . "\n";

// 2 秒待つ
usleep(2000000);

// 復帰!
echo date('h:i:s') . "\n";

?>

上の例の出力は以下となります。

11:13:28
11:13:30

参考

  • time_sleep_until() - 指定した時刻まで実行を遅延する
