SplFileObject::getChildren

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

SplFileObject::getChildren — 用途はなし

説明

public SplFileObject::getChildren(): void

SplFileObject は子を持たないのでこのメソッドは null を返します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
