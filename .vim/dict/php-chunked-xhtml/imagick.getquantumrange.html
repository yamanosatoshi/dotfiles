Imagick::getQuantumRange

(PECL imagick 2, PECL imagick 3)

Imagick::getQuantumRange — Imagick quantum range を返す

説明

public static Imagick::getQuantumRange(): array

Imagick インスタンスの quantum range を文字列で返します。

パラメータ

この関数にはパラメータはありません。

返り値

quantum range を連想配列で返します。 int ("quantumRangeLong") と string ("quantumRangeString") が含まれます。

エラー / 例外

エラー時に ImagickException をスローします。
