putenv

(PHP 4, PHP 5, PHP 7, PHP 8)

putenv — 環境変数の値を設定する

説明

putenv(string $assignment): bool

サーバーの環境変数に assignment を追加します。 この環境変数は、カレントのリクエストを実行している間のみ存在します。 リクエスト終了時、環境変数は元の状態に戻されます。

パラメータ

assignment

"FOO=BAR" 形式の設定。

返り値

成功した場合に true を、失敗した場合に false を返します。

例

例1 環境変数の設定

<?php
putenv("UNIQID=$uniqid");
?>
