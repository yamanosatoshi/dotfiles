DOMNode::cloneNode

(PHP 5, PHP 7, PHP 8)

DOMNode::cloneNode — ノードを複製する

説明

public DOMNode::cloneNode(bool $deep = false): DOMNode|false

ノードのコピーを作成します。

パラメータ

deep

子孫要素を含めてコピーするかどうかを指定します。 このパラメータのデフォルト値は false です。

返り値

複製されたノードを返します。
