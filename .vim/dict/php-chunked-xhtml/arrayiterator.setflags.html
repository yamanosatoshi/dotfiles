ArrayIterator::setFlags

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

ArrayIterator::setFlags — 振る舞いのフラグを設定する

説明

public ArrayIterator::setFlags(string $flags): void

振る舞いのフラグを設定します。

パラメータ

flags

新しい ArrayIterator の振る舞い。 以下のいずれかの値のビットマスクか、名前付き定数を設定します。 将来のバージョンとの互換性を保証するため、 名前付き定数を使うことを強く推奨します。

利用可能なフラグは以下のとおりです。 これらのフラグの実際の意味は、 定義済みの定数 で説明しています。

ArrayIterator の振る舞いを表すフラグ
値 定数
1 ArrayIterator::STD_PROP_LIST
2 ArrayIterator::ARRAY_AS_PROPS

返り値

値を返しません。
