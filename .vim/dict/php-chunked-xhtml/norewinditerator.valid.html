NoRewindIterator::valid

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

NoRewindIterator::valid — イテレータの有効性を検証する

説明

public NoRewindIterator::valid(): bool

イテレータが有効かどうかを調べます。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に true を、失敗した場合に false を返します。
