mysqli_get_links_stats

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

mysqli_get_links_stats — オープン済みとキャッシュ済みのリンクに関する情報を返す

説明

mysqli_get_links_stats(): array

mysqli_get_links_stats() 関数は、 オープン済みとキャッシュ済みの MySQL リンクに関する情報を返します。

パラメータ

この関数にはパラメータはありません。

返り値

mysqli_get_links_stats() 関数は、3つの要素からなる連想配列を返します。 キーは次のとおりです:

total

オープンされているリンクの合計数を示す int

active_plinks

アクティブな永続的な接続の数を示す int

cached_plinks

アクティブでない永続的な接続の数を示す int
