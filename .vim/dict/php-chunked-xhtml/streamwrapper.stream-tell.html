streamWrapper::stream_tell

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

streamWrapper::stream_tell — ストリームの現在位置を取得する

説明

public streamWrapper::stream_tell(): int

このメソッドは、fseek() に対応してコールされ、現在の位置を決定します。

パラメータ

この関数にはパラメータはありません。

返り値

ストリームの現在位置を返します。
