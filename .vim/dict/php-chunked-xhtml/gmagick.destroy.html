Gmagick::destroy

(PECL gmagick >= Unknown)

Gmagick::destroy — Gmagick オブジェクトを破棄する

説明

public Gmagick::destroy(): bool

Gmagick オブジェクトを破棄し、関連するリソースをすべて解放します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に true を、失敗した場合に false を返します。

エラー / 例外

エラー時に GmagickException をスローします。
