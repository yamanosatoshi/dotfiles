Gmagick::getimageblueprimary

(PECL gmagick >= Unknown)

Gmagick::getimageblueprimary — 青が一番強い点を返す

説明

public Gmagick::getimageblueprimary(): array

青が一番強い点を返します。

パラメータ

x

青が一番強い点の x 座標。

y

青が一番強い点の y 座標。

返り値

"x" 座標および "y" 座標を含む配列を返します。

エラー / 例外

エラー時に GmagickException をスローします。
