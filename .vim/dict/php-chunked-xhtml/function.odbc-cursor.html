odbc_cursor

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_cursor — カーソル名を得る

説明

odbc_cursor(resource $statement): string|false

指定された接続 ID に関するカーソル名を返します。 Gets the cursorname for the given result_id.

パラメータ

statement

結果 ID。

返り値

カーソル名を文字列で返します。 失敗した場合に false を返します
