ldap_next_entry

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_next_entry — 次の結果エントリを得る

説明

ldap_next_entry(resource $ldap, resource $entry): resource|false

結果に保持されたエントリを取得します。 連続的に ldap_next_entry() をコールした場合、 エントリがなくなるまでエントリを一つずつ返します。 ldap_next_entry() への最初のコールは、 ldap_first_entry() から返された entry を指定してコールした後に、その結果を用いて行います。

パラメータ

ldap

ldap_connect() が返す LDAP リソース。

entry

返り値

ldap_first_entry() によりエントリが読み始められた 結果において次のエントリに関するエントリID を返します。 これ以上結果のエントリがない場合は、falseを返します。
