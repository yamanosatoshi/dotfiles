ZipArchive::getStatusString

(PHP 5 >= 5.2.7, PHP 7, PHP 8)

ZipArchive::getStatusString — システムエラー、または ZIP メッセージに関するエラー情報を返す

説明

public ZipArchive::getStatusString(): string

システムエラー、または ZIP メッセージに関するエラー情報を返す

パラメータ

この関数にはパラメータはありません。

返り値

成功時にはステータスメッセージを文字列で返します。

変更履歴

バージョン 説明
8.0.0 / 1.18.0 このメソッドは、閉じられたアーカイブに対しても呼び出せるようになりました。
8.0.0 / 1.18.0 このメソッドは、失敗時に false を返さなくなりました。
