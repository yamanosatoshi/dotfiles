Yaf_Request_Abstract::setActionName

(Yaf >=1.0.0)

Yaf_Request_Abstract::setActionName — アクション名を設定する

説明

public Yaf_Request_Abstract::setActionName(string $action, bool $format_name = true): void

アクション名をリクエストに設定します。 このメソッドは通常、ルーティング結果のコントローラー名を設定するカスタムルータで使います。

パラメータ

action

アクション名を表すstring。 "index" や "foo_Bar" のように、小文字で表記すべきです。

format_name

Yaf 3.2.0 で追加されました。 デフォルトでは、Yaf は名前を小文字に変換しますが、 このパラメータを false に設定すると、指定された値そのままをリクエストに使います。

返り値
