jdtojulian

(PHP 4, PHP 5, PHP 7, PHP 8)

jdtojulian — ユリウス積算日をユリウス暦に変換する

説明

jdtojulian(int $julian_day): string

ユリウス積算日を "月/日/年" の形式でユリウス暦を含む文字列に変換します。

パラメータ

julian_day

ユリウス積算日を表す整数値。

返り値

ユリウス暦の日付を "月/日/年" 形式の文字列で返します。
