mysqli::release_savepoint

mysqli_release_savepoint

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

mysqli::release_savepoint -- mysqli_release_savepoint — 現在のトランザクションのセーブポイント群から、指定した名前のセーブポイントを削除する

説明

オブジェクト指向型

public mysqli::release_savepoint(string $name): bool

手続き型:

mysqli_release_savepoint(mysqli $mysql, string $name): bool

この関数は、 $mysqli->query("RELEASE SAVEPOINT `$name`"); を実行することと同等です。 この関数は、コミットやロールバックを実行しません。

パラメータ

link

手続き型のみ: mysqli_connect() あるいは mysqli_init() が返す mysqliオブジェクト。

name

セーブポイントの識別子。

返り値

成功した場合に true を、失敗した場合に false を返します。
