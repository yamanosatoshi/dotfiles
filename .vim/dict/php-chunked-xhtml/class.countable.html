Countable インターフェイス

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

はじめに

Countable を実装したクラスは、 count() 関数で使用することができます。

インターフェイス概要

Countable {
/* メソッド */
abstract public count(): int
}
