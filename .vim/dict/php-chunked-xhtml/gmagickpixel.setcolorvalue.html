GmagickPixel::setcolorvalue

(PECL gmagick >= Unknown)

GmagickPixel::setcolorvalue — 色チャネルの正規化した値を設定する

説明

public GmagickPixel::setcolorvalue(int $color, float $value): GmagickPixel

このオブジェクトの指定したチャネルの値を設定します。これは 0 から 1 までの間の値になります。 この関数を使って GmagickPixel オブジェクトの透明度チャネルを設定できます。

パラメータ

color

Gmagick チャネル定数のいずれか。

value

設定したい値。0 から 1 までの範囲で指定します。

返り値

成功した場合に GmagickPixel オブジェクトを返します。
