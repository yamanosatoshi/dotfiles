quotemeta

(PHP 4, PHP 5, PHP 7, PHP 8)

quotemeta — メタ文字をクォートする

説明

quotemeta(string $string): string

文字列 str について、

. \ + * ? [ ^ ] ( $ )
の前にバックスラッシュ文字 (\) でクォートして返します。

パラメータ

string

入力文字列。

返り値

メタ文字をクォートした文字列を返します。 空文字を string に渡した場合は false を返します。

注意

注意: この関数はバイナリデータに対応しています。
