get_magic_quotes_gpc

(PHP 4, PHP 5, PHP 7)

get_magic_quotes_gpc — magic_quotes_gpc の現在の設定を得る

警告

この関数は PHP 7.4.0 で 非推奨 になり、PHP 8.0.0 で 削除 されました。この関数に頼らないことを強く推奨します。

説明

get_magic_quotes_gpc(): bool

常に false を返します。

パラメータ

この関数にはパラメータはありません。

返り値

常に false を返します。

変更履歴

バージョン 説明
7.4.0 この関数は推奨されなくなりました。
