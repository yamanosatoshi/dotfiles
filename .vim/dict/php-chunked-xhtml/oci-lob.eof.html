OCILob::eof

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCILob::eof — ラージオブジェクトのディスクリプタが EOF かどうかを調べる

説明

OCILob::eof(): bool

ラージオブジェクトの内部ポインタが LOB の最後にあるかどうかを調べます。

パラメータ

この関数にはパラメータはありません。

返り値

もし、ラージオブジェクトの内部ポインタが LOB の最後にあるとき、 true を返します。その他の場合は false を返します。

変更履歴

バージョン 説明
8.0.0, PECL OCI8 3.0.0 OCI-Lob クラスは、 PHP の名前付けの規約に従って OCILob に名前が変更されました。

注意

注意:

OCILob::setBuffering が LOB で有効になっていない場合は、Oracle のエラーを返します。
