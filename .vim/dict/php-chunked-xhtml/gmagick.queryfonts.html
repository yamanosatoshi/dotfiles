Gmagick::queryfonts

(PECL gmagick >= Unknown)

Gmagick::queryfonts — 設定したフォントを返す

説明

public Gmagick::queryfonts(string $pattern = "*"): array

設定したフォントを返します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
