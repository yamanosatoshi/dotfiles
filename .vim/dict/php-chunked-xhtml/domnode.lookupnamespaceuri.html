DOMNode::lookupNamespaceUri

(PHP 5, PHP 7, PHP 8)

DOMNode::lookupNamespaceUri — プレフィックスに基づいて、ノードの名前空間 URI を取得する

説明

public DOMNode::lookupNamespaceUri(string $prefix): string

prefix に基づいて、 ノードの名前空間 URI を取得します。

パラメータ

prefix

名前空間のプレフィックス。

返り値

ノードの名前空間 URI を返します。
