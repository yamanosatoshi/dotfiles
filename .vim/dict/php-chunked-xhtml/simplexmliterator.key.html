SimpleXMLIterator::key

(PHP 5, PHP 7, PHP 8)

SimpleXMLIterator::key — 現在のキーを返す

説明

public SimpleXMLIterator::key(): mixed

このメソッドは、現在の要素の XML タグ名を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の SimpleXMLIterator オブジェクトが参照する要素の XML タグ名、あるいは false を返します。

例

例1 現在の XML タグキーの取得

<?php
$xmlIterator = new SimpleXMLIterator('<books><book>PHP basics</book><book>XML basics</book></books>');

echo var_dump($xmlIterator->key());
$xmlIterator->rewind(); // 最初の要素に巻き戻します
echo var_dump($xmlIterator->key());

?>

上の例の出力は以下となります。

bool(false)
string(4) "book"
