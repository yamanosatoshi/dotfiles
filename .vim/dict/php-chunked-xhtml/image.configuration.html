実行時設定

php.ini の設定により動作が変化します。

Image 設定オプション
名前 デフォルト 変更可能 変更履歴
gd.jpeg_ignore_warning "1" PHP_INI_ALL  
PHP_INI_* モードの詳細および定義については どこで設定を行うのか を参照してください。

以下に設定ディレクティブに関する 簡単な説明を示します。

gd.jpeg_ignore_warning bool

libjpeg(-turbo) が出す警告（エラーではない）を無視します。

gd.jpeg_ignore_warning の変更履歴
バージョン 説明
7.1.0 gd.jpeg_ignore_warning のデフォルト値が 0 から 1 に変わりました。

exif の設定ディレクティブも参照ください。

警告

イメージ関数は非常にメモリを消費します。 バンドルされたバージョンの GD ライブラリを使用している場合、 memory_limit を十分大きな値にしておくようにしましょう。
