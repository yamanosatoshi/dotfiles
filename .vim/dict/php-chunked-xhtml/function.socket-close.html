socket_close

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

socket_close — Socket クラスのインスタンスを閉じる

説明

socket_close(Socket $socket): void

socket_close() は、 socket で指定した Socket クラスのインスタンスを閉じます。

パラメータ

socket

socket_create() あるいは socket_accept() で作成した Socket クラスのインスタンス。

返り値

値を返しません。

変更履歴

バージョン 説明
8.0.0 socket は、Socket クラスのインスタンスになりました。 これより前のバージョンでは、リソース型でした。
