imap_msgno

(PHP 4, PHP 5, PHP 7, PHP 8)

imap_msgno — 指定した UID のメッセージ番号を返す

説明

imap_msgno(resource $imap, int $message_uid): int

指定した message_uid のメッセージ番号を返します。

この関数は、imap_uid() の逆の動作となります。

パラメータ

imap

imap_open() が返す IMAP ストリーム。

message_uid

メッセージの UID。

返り値

指定した message_uid のメッセージ番号を返します。
