gmp_root

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

gmp_root — n乗根の整数部を返す

説明

gmp_root(GMP|int|string $num, int $nth): GMP

num の nth 乗根を計算し、 その整数部を返します。

パラメータ

num

GMP オブジェクト、整数、あるいは数値に変換可能な数値形式の文字列。

nth

num の正のべき乗根

返り値

結果の整数部をGMP数として返します。
