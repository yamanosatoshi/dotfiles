Gmagick::despeckleimage

(PECL gmagick >= Unknown)

Gmagick::despeckleimage — 画像内のスペックルノイズを軽減する

説明

public Gmagick::despeckleimage(): Gmagick

画像内のスペックルノイズを軽減します。ただし元画像の輪郭は保持します。

パラメータ

この関数にはパラメータはありません。

返り値

スペックルノイズを軽減した Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
