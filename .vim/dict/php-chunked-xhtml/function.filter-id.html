filter_id

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

filter_id — フィルタの名前からフィルタ ID を返す

説明

filter_id(string $name): int|false

パラメータ

name

取得したいフィルタの名前。

返り値

フィルタの ID を返します。フィルタが存在しない場合は false を返します。
