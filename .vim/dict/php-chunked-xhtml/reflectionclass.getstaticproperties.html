ReflectionClass::getStaticProperties

(PHP 5, PHP 7, PHP 8)

ReflectionClass::getStaticProperties — 静的なプロパティを取得する

説明

public ReflectionClass::getStaticProperties(): array|null

静的なプロパティを取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

静的なプロパティを配列で返します。 失敗した場合は、null を返します。
