Imagick::getImageSize

(PECL imagick 2, PECL imagick 3)

Imagick::getImageSize — 画像の長さをバイト数で返す

警告

この関数は Imagick 3.4.4 以降では 非推奨 になりました。この関数に依存したコードを書くのはやめましょう。

説明

public Imagick::getImageSize(): int

画像の長さをバイト数で返します。 非推奨です。かわりに Imagick::getImageLength() を使いましょう。

パラメータ

この関数にはパラメータはありません。

返り値

現在の画像のサイズをバイト数で返します。
