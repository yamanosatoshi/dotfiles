gmp_rootrem

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

gmp_rootrem — n乗根の整数部分と小数部分を返す

説明

gmp_rootrem(GMP|int|string $num, int $nth): array

num の nth 乗根を計算し、 整数部分と小数部分を返します。

パラメータ

num

GMP オブジェクト、整数、あるいは数値に変換可能な数値形式の文字列。

nth

num の正のn乗根

返り値

ふたつの要素の配列を返します。 最初の要素はn乗根の整数部分、ふたつめの要素は小数部分です。 両方ともGMP数として表現されます。
