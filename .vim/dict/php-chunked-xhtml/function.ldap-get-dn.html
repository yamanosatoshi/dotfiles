ldap_get_dn

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_get_dn — 結果エントリから DN を得る

説明

ldap_get_dn(resource $ldap, resource $entry): string|false

結果における、あるエントリの DN を見つけます。

パラメータ

ldap

ldap_connect() が返す LDAP リソース。

entry

返り値

結果エントリの DN を返します。エラーの場合、falseを返します。
