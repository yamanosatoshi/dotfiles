SplDoublyLinkedList::serialize

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

SplDoublyLinkedList::serialize — ストレージをシリアライズする

説明

public SplDoublyLinkedList::serialize(): string

ストレージをシリアライズします。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

シリアライズした文字列を返します。
