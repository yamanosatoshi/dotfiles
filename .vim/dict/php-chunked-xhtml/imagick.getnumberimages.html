Imagick::getNumberImages

(PECL imagick 2, PECL imagick 3)

Imagick::getNumberImages — オブジェクト内の画像の数を返す

説明

public Imagick::getNumberImages(): int

Imagick オブジェクトに関連付けられている画像の数を返します。

パラメータ

この関数にはパラメータはありません。

返り値

Imagick オブジェクトに関連付けられている画像の数を返します。

エラー / 例外

エラー時に ImagickException をスローします。
