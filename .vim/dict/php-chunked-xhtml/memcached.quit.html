Memcached::quit

(PECL memcached >= 2.0.0)

Memcached::quit — 開いている接続を閉じる

説明

public Memcached::quit(): bool

Memcached::quit() は、memcache サーバーとの接続をすべて閉じます。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に true を、失敗した場合に false を返します。
