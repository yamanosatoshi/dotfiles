Imagick::animateImages

(PECL imagick 2 >= 2.3.0, PECL imagick 3)

Imagick::animateImages — 画像を動かす

説明

public Imagick::animateImages(string $x_server): bool

このメソッドは、ローカルあるいはリモートの X サーバーで画像を動かします。 このメソッドは Windows では使えません。 このメソッドは、ImageMagick バージョン 6.3.6 以降で Imagick をコンパイルした場合に使用可能です。

パラメータ

x_server

X サーバーのアドレス。

返り値

成功した場合に true を返します。
