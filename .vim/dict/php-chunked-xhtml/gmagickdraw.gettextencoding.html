GmagickDraw::gettextencoding

(PECL gmagick >= Unknown)

GmagickDraw::gettextencoding — テキストのアノテーションで使うコードセットを返す

説明

public GmagickDraw::gettextencoding(): mixed

テキストのアノテーションで使うコードセットを表す文字列を返します。

パラメータ

この関数にはパラメータはありません。

返り値

コードセットを表す文字列を返します。 エンコーディングが設定されていない場合は false を返します。
