fbsql_num_fields

(PHP 4 >= 4.0.6, PHP 5 < 5.3.0)

fbsql_num_fields — 結果のフィールド数を得る

説明

fbsql_num_fields(resource $result): int

指定した結果セット result のフィールド数を返します。

パラメータ

result

fbsql_query() あるいは fbsql_db_query() が返す結果 ID。

返り値

フィールドの数を整数値で返します。
