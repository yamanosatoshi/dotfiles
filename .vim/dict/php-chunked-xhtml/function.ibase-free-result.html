ibase_free_result

(PHP 5, PHP 7 < 7.4.0)

ibase_free_result — 結果セットを解放する

説明

ibase_free_result(resource $result_identifier): bool

結果セットを解放します。

パラメータ

result_identifier

ibase_query() あるいは ibase_execute() で作成した結果セット。

返り値

成功した場合に true を、失敗した場合に false を返します。
