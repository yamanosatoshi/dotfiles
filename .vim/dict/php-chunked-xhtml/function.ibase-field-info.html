ibase_field_info

(PHP 5, PHP 7 < 7.4.0)

ibase_field_info — フィールドに関する情報を得る

説明

ibase_field_info(resource $result, int $field_number): array

select クエリが実行された後、フィールドに関する情報を配列として返します。

パラメータ

result

InterBase 結果 ID。

field_number

フィールドのオフセット。

返り値

配列を返します。配列に含まれるキーは name、 alias、relation、 length そして type です。

例

例1 ibase_field_info() の例

<?php
$rs = ibase_query("SELECT * FROM tablename");
$coln = ibase_num_fields($rs);
for ($i = 0; $i < $coln; $i++) {
    $col_info = ibase_field_info($rs, $i);
    echo "name: ". $col_info['name']. "\n";
    echo "alias: ". $col_info['alias']. "\n";
    echo "relation: ". $col_info['relation']. "\n";
    echo "length: ". $col_info['length']. "\n";
    echo "type: ". $col_info['type']. "\n";
}
?>
