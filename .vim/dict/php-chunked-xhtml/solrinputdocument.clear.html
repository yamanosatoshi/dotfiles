SolrInputDocument::clear

(PECL solr >= 0.9.2)

SolrInputDocument::clear — 入力した文書をリセット

説明

public SolrInputDocument::clear(): bool

全フィールドをドロップして文書をリセットし、文書の boost をゼロにリセットします。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に true を、失敗した場合に false を返します。
