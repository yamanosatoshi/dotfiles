Imagick::getImageExtrema

(PECL imagick 2, PECL imagick 3)

Imagick::getImageExtrema — 画像の極値を取得する

警告

この関数は Imagick 3.4.4 以降では 非推奨 になりました。この関数に依存したコードを書くのはやめましょう。

説明

public Imagick::getImageExtrema(): array

画像の極値を取得します。 "min" および "max" というキーを持つ連想配列を返します。

パラメータ

この関数にはパラメータはありません。

返り値

"min" および "max" というキーを持つ連想配列を返します。

エラー / 例外

エラー時に ImagickException をスローします。
