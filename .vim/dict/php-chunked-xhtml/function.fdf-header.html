fdf_header

(PHP 4 >= 4.3.0, PHP 5 < 5.3.0, PECL fdf SVNf)

fdf_header — FDF 固有の出力ヘッダをセットする

説明

fdf_header(): void

この関数は、簡便に FDF の出力に適当な HTTP ヘッダを付加できるよう 用意されており、Content-type: を application/vnd.fdf に設定します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
