pg_flush

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

pg_flush — 送信を待っているクエリデータをフラッシュする

説明

pg_flush(resource $connection): mixed

pg_flush() は、送信を待機中のすべての クエリデータを接続へフラッシュします。

パラメータ

connection

PostgreSQL の接続リソース。

返り値

フラッシュが成功したか、またはフラッシュを待機中のデータがなかった場合 true、 保留中のデータの一部がフラッシュされたが、まだ残りがある場合、0。 失敗した場合に false を返します
