注意

注意:

DOM拡張機能 は UTF-8 エンコーディングを使います。ISO-8859-1 エンコーディングのテキストを扱う場合は utf8_encode() と utf8_decode() を使ってください。ISO-8859-1 以外のエンコーディングの場合は iconv を使ってください。
