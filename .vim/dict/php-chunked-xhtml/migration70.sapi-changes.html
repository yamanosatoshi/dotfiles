SAPI モジュールの変更

FPM

制限のない listen ポートは、 IPv4 と IPv6 の両方をリスンするようになりました。

PHP 5 では、listen ディレクティブにポート番号だけを指定した場合は、 すべてのインターフェイス上で IPv4 だけをリスンしていました。PHP 7 では、 IPv4 と IPv6 の両方のリクエストを受け付けるようになります。

特定の IP アドレスを指定しているディレクティブには何も影響を及ぼしません。 IP アドレスを指定している場合は、そのアドレスとプロトコルだけを受け付けます。
