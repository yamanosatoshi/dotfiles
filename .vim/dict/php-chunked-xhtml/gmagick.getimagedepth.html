Gmagick::getimagedepth

(PECL gmagick >= Unknown)

Gmagick::getimagedepth — 画像の深度を取得する

説明

public Gmagick::getimagedepth(): int

画像の深度を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

画像の深度を返します。

エラー / 例外

エラー時に GmagickException をスローします。
