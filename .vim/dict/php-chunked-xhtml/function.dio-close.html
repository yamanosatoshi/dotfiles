dio_close

(PHP 4 >= 4.2.0, PHP 5 < 5.1.0)

dio_close — fd で指定したファイル記述子を閉じる

説明

dio_close(resource $fd): void

関数 dio_close() は、ファイル記述子 fd を閉じます。

パラメータ

fd

dio_open() が返すファイル記述子。

返り値

値を返しません。

例

例1 開いているファイル記述子を閉じる

<?php
$fd = dio_open('/dev/ttyS0', O_RDWR);

dio_close($fd);
?>
