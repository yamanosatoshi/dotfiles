gnupg_cleardecryptkeys

(PECL gnupg >= 0.5)

gnupg_cleardecryptkeys — 事前に復号のために設定されたすべてのキーを削除する

説明

gnupg_cleardecryptkeys(resource $identifier): bool

パラメータ

identifier

gnupg_init() あるいは gnupg のコールで得られた gnupg ID。

返り値

成功した場合に true を、失敗した場合に false を返します。

例

例1 手続き型の gnupg_cleardecryptkeys() の例

<?php
$res = gnupg_init();
gnupg_cleardecryptkeys($res);
?>

例2 オブジェクト指向の gnupg_cleardecryptkeys() の例

<?php
$gpg = new gnupg();
$gpg -> cleardecryptkeys();
?>
