ps_fill

(PECL ps >= 1.1.0)

ps_fill — 現在のパスを塗りつぶす

説明

ps_fill(resource $psdoc): bool

ps_lineto() のような描画関数によって事前に作成されたパスを塗りつぶします。

パラメータ

psdoc

ps_new() が返す、postscript ファイルのリソース ID。

返り値

成功した場合に true を、失敗した場合に false を返します。
