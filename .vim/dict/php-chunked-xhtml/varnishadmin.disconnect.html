VarnishAdmin::disconnect

(PECL varnish >= 1.0.0)

VarnishAdmin::disconnect — varnish インスタンスの管理インターフェイスとの接続を切断する

説明

public VarnishAdmin::disconnect(): bool

パラメータ

この関数にはパラメータはありません。
