Imagick::previousImage

(PECL imagick 2, PECL imagick 3)

Imagick::previousImage — オブジェクト内の前の画像に移動する

説明

public Imagick::previousImage(): bool

画像リスト内の前の画像を Imagick オブジェクトに関連付けます。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に true を返します。
