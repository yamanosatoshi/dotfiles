imap_base64

(PHP 4, PHP 5, PHP 7, PHP 8)

imap_base64 — BASE64 でエンコードされたテキストをデコードする

説明

imap_base64(string $string): string|false

BASE64 でエンコードされたテキスト string をデコードします。

パラメータ

string

エンコードされたテキスト。

返り値

デコードしたメッセージを文字列で返します。 失敗した場合に false を返します
