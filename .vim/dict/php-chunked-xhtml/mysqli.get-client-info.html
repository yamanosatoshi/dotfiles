mysqli::$client_info

mysqli::get_client_info

mysqli_get_client_info

(PHP 5, PHP 7, PHP 8)

mysqli::$client_info -- mysqli::get_client_info -- mysqli_get_client_info — MySQL クライアント情報を取得する

説明

オブジェクト指向型

string|null $mysqli->client_info;
public mysqli::get_client_info(): string|null

手続き型

mysqli_get_client_info(mysqli|null $mysql = null): string|null

MySQL クライアントライブラリのバージョンを文字列で返します。

パラメータ

この関数にはパラメータはありません。

返り値

MySQL クライアントライブラリのバージョンを文字列で返します。

例

例1 mysqli_get_client_info

<?php

/* mysql クライアントライブラリのバージョンを
   調べるために、接続する必要はありません */

printf("Client library version: %s\n", mysqli_get_client_info());
?>
