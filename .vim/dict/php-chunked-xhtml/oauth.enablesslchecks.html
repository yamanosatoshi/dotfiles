OAuth::enableSSLChecks

(PECL OAuth >= 0.99.5)

OAuth::enableSSLChecks — SSL チェックを有効にする

説明

public OAuth::enableSSLChecks(): bool

通常の SSL ピア証明書およびホストのチェックを有効にします (デフォルトは有効です)。 あるいは、sslChecks メンバーに false 以外の値を設定することによっても SSL チェックを有効にできます。

パラメータ

この関数にはパラメータはありません。

返り値

true

変更履歴

バージョン 説明
PECL oauth 0.99.8 sslChecks メンバーが追加されました。
