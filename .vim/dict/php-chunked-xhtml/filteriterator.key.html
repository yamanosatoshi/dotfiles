FilterIterator::key

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

FilterIterator::key — 現在のキーを取得する

説明

public FilterIterator::key(): mixed
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

現在のキーを取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在のキーを返します。
