ldap_unbind

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_unbind — LDAP ディレクトリへのバインドを解除する

説明

ldap_unbind(resource $ldap): bool

LDAP ディレクトリへのバインドを解除します。

パラメータ

ldap

ldap_connect() が返す LDAP リソース。

返り値

成功した場合に true を、失敗した場合に false を返します。
