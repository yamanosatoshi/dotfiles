eio_grp_cancel

(PECL eio >= 0.0.1dev)

eio_grp_cancel — リクエストグループをキャンセルする

説明

eio_grp_cancel(resource $grp): void

eio_grp_cancel() は、 grp で指定したグループリクエストをキャンセルします。

パラメータ

grp

eio_grp() が返すリクエストグループリソース。

返り値

値を返しません。
