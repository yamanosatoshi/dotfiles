Imagick::setCompression

(PECL imagick 2, PECL imagick 3)

Imagick::setCompression — オブジェクトのデフォルトの圧縮方式を設定する

説明

public Imagick::setCompression(int $compression): bool

オブジェクトのデフォルトの圧縮方式を設定します。

パラメータ

compression

圧縮方式。Imagick::COMPRESSION_* 定数を参照ください。

返り値

成功した場合に true を返します。
