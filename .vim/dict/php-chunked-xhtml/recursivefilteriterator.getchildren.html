RecursiveFilterIterator::getChildren

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

RecursiveFilterIterator::getChildren — RecursiveFilterIterator に含まれる内部イテレータの子を返す

説明

public RecursiveFilterIterator::getChildren(): RecursiveFilterIterator

RecursiveFilterIterator に含まれる内部イテレータの子を返します。

パラメータ

この関数にはパラメータはありません。

返り値

RecursiveFilterIterator に含まれる内部イテレータの子を返します。
