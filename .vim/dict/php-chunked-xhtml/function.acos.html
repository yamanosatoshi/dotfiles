acos

(PHP 4, PHP 5, PHP 7, PHP 8)

acos — 逆余弦（アークコサイン）

説明

acos(float $num): float

num のアークコサインをラジアンで返します。 acos() は cos() の逆関数で、 acos() がとりうる範囲内のすべての a について a==cos(acos(a)) が成立します。

パラメータ

num

処理する角度。

返り値

num のアークコサインをラジアンで返します。
