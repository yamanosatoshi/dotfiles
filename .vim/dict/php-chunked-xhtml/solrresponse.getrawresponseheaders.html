SolrResponse::getRawResponseHeaders — Solr サーバーからの生のレスポンスヘッダを返す

説明

public SolrResponse::getRawResponseHeaders(): string

Solr サーバーからの生のレスポンスヘッダを返します。

この関数にはパラメータはありません。
