exif_tagname

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

exif_tagname — インデックスに対応するヘッダ名を取得する

説明

exif_tagname(int $index): string|false

パラメータ

index

名前を調べたいタグの ID。

返り値

ヘッダ名を返します。もし index が EXIF タグ ID として未定義の場合は false を返します。

例

例1 exif_tagname() の例

<?php
echo "256: ".exif_tagname(256).PHP_EOL;
echo "257: ".exif_tagname(257).PHP_EOL;
?>

上の例の出力は以下となります。

256: ImageWidth
257: ImageLength
