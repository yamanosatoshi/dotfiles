Windows 上での PHP のトラブルシューティング

テンポラリディレクトリのパーミッションのチェック

  1. エクスプローラでテンポラリディレクトリを右クリックして、パーミッションを調べます。

  2. IIS の場合は、 IIS_User ユーザーがこのディレクトリへの書き込み権限を持っている必要があります。 テンポラリディレクトリの場所は、php の設定や phpinfo で取得できます。
