imagetypes

(PHP 4 >= 4.0.2, PHP 5, PHP 7, PHP 8)

imagetypes — この PHP がサポートしている画像形式を返す

説明

imagetypes(): int

現在使用している PHP がサポートする画像形式を返します。

パラメータ

この関数にはパラメータはありません。

返り値

画像形式に対応するビットフィールドで、 PHP に組み込まれている GD がサポートする画像形式を返します。 返されるビットは次のとおりです。 IMG_BMP | IMG_GIF | IMG_JPG | IMG_PNG | IMG_WBMP | IMG_XPM | IMG_WEBP。

変更履歴

バージョン 説明
7.2.0 IMG_BMP が追加されました。
7.0.10 IMG_WEBP が追加されました。

例

例1 PNG をサポートしているかどうかの確認

<?php
if (imagetypes() & IMG_PNG) {
    echo "PNG をサポートしています";
}
?>
