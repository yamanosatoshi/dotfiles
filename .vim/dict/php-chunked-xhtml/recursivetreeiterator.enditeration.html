RecursiveTreeIterator::endIteration

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

RecursiveTreeIterator::endIteration — 反復処理を終了する

説明

public RecursiveTreeIterator::endIteration(): void

反復処理が終了したとき (RecursiveTreeIterator::valid() が最初に false を返したとき) にコールされます。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
