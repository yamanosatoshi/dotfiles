GmagickDraw::ellipse

(PECL gmagick >= Unknown)

GmagickDraw::ellipse — 楕円を画像上に描く

説明

public GmagickDraw::ellipse(
    float $ox,
    float $oy,
    float $rx,
    float $ry,
    float $start,
    float $end
): GmagickDraw

楕円を画像上に描きます。

パラメータ

ox

原点の x 座標。

oy

原点の y 座標。

rx

x 方向の半径。

ry

y 方向の半径。

start

開始角度。

end

終了角度。

返り値

成功した場合に GmagickDraw オブジェクトを返します。
