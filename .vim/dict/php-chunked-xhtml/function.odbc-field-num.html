odbc_field_num

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_field_num — カラム番号を返す

説明

odbc_field_num(resource $statement, string $field): int|false

指定したODBC 結果 ID におけるフィールド名に対応するカラムスロットの数を取得します。

パラメータ

statement

結果 ID。

field

フィールド名。

返り値

フィールド番号、あるいはエラー時に false を返します。 フィールド番号は、1 から始まります。
