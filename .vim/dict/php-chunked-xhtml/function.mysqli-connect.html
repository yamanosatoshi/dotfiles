mysqli_connect

(PHP 5, PHP 7, PHP 8)

mysqli_connect — mysqli::__construct() のエイリアス

説明

この関数は次の関数のエイリアスです。 mysqli::__construct()

注意:

mysqli の例外モードが無効な場合、かつ接続に失敗した場合、 mysqli_connect() 関数はオブジェクトではなく false を返します。 接続エラーの内容を取得するのに、 mysqli_connect_error() 関数が使えます。
