zlib_get_coding_type

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

zlib_get_coding_type — 出力圧縮に使用されたコーディングの種類を返す

説明

zlib_get_coding_type(): string|false

出力圧縮に使用されたコーディングの種類を返します。

パラメータ

この関数にはパラメータはありません。

返り値

返される値は gzip, deflate, もしくは false です。
