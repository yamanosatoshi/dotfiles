OCILob::size

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCILob::size — ラージオブジェクトのサイズを返す

説明

OCILob::size(): int|false

ラージオブジェクトのサイズを取得します。

パラメータ

この関数にはパラメータはありません。

返り値

ラージオブジェクトのサイズを返します。 失敗した場合に false を返します。 空のオブジェクトは長さ 0 となります。

変更履歴

バージョン 説明
8.0.0, PECL OCI8 3.0.0 OCI-Lob クラスは、 PHP の名前付けの規約に従って OCILob に名前が変更されました。
