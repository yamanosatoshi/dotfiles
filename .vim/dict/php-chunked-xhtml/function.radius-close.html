radius_close

(PECL radius >= 1.1.0)

radius_close — すべてのリソースを開放する

説明

radius_close(resource $radius_handle): bool

各リクエストの終了時に PHP がすべてのリソースを開放するので、 この関数をコールする必要はありません。

パラメータ

radius_handle

RADIUS リソース。

返り値

成功した場合に true を、失敗した場合に false を返します。
