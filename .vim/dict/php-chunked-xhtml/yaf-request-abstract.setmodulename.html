Yaf_Request_Abstract::setModuleName

(Yaf >=1.0.0)

Yaf_Request_Abstract::setModuleName — モジュール名を設定する

説明

public Yaf_Request_Abstract::setModuleName(string $module, bool $format_name = true): void

リクエストするモジュール名を設定します。 このメソッドは通常、ルーティング結果のモジュール名を設定するカスタムルータで使います。

パラメータ

module

モジュール名を表すstring。 "Index" や "Foo_Bar" のように、キャメルケースで表記すべきです。

format_name

Yaf 3.2.0 で追加されました。 デフォルトでは、Yaf は名前をキャメルケースでフォーマットしますが、 このパラメータを false に設定すると、指定された値そのままをリクエストに使います。

返り値
