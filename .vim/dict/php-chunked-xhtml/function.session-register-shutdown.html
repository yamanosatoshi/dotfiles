session_register_shutdown

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

session_register_shutdown — セッションのシャットダウン関数

説明

session_register_shutdown(): void

session_write_close() をシャットダウン関数として登録します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

エラー / 例外

シャットダウン関数の登録に失敗した場合に E_WARNING が発生します。
