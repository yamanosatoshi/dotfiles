asin

(PHP 4, PHP 5, PHP 7, PHP 8)

asin — 逆正弦（アークサイン）

説明

asin(float $num): float

num のアークサインをラジアンで返します。 asin() は sin() の逆関数で、 asin() がとりうる範囲内のすべての a について a==sin(asin(a)) が成立します。

パラメータ

num

処理する角度。

返り値

num のアークサインをラジアンで返します。

  • asinh() - 逆双曲線正弦（アークハイパボリックサイン）
  • acos() - 逆余弦（アークコサイン）
  • atan() - 逆正接（アークタンジェント）
