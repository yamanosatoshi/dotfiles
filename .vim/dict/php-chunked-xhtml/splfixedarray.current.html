SplFixedArray::current

(PHP 5 >= 5.3.0, PHP 7)

SplFixedArray::current — 現在の配列の要素を返す

説明

public SplFixedArray::current(): mixed

現在の配列の要素を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の要素の値を返します。

エラー / 例外

内部の配列ポインタが無効なインデックスを指していたり、範囲外にあったりする場合に RuntimeException をスローします。
