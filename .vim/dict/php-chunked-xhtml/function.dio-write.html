dio_write

(PHP 4 >= 4.2.0, PHP 5 < 5.1.0)

dio_write — オプションで丸め長さを指定してデータを書き込む

説明

dio_write(resource $fd, string $data, int $len = 0): int

dio_write() は、data から最大 len バイトを ファイル fd に書き込みます。

パラメータ

fd

dio_open() が返すファイル記述子。

data

書き込むデータ。

len

書き込むデータのバイト長。指定しなかった場合は、データ全体を 指定したファイルに書き込みます。

返り値

fd に書き込んだバイト数を返します。
