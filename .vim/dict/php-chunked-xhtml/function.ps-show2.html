ps_show2

(PECL ps >= 1.1.0)

ps_show2 — テキストを現在の位置に出力する

説明

ps_show2(resource $psdoc, string $text, int $len): bool

テキストを現在の位置に出力します。最大 len 文字までしか出力しません。

パラメータ

psdoc

ps_new() が返す postscript ファイルの ID リソース。

text

出力するテキスト。

len

表示する最大文字数。

返り値

成功した場合に true を、失敗した場合に false を返します。
