DOMNode::isSameNode

(PHP 5, PHP 7, PHP 8)

DOMNode::isSameNode — 2 つのノードが等しいかどうかを調べる

説明

public DOMNode::isSameNode(DOMNode $otherNode): bool

この関数は、2 つのノードが等しいかどうかを調べます。この比較は、 その内容に基づくものではありません。

パラメータ

otherNode

比較対象となるノード。

返り値

成功した場合に true を、失敗した場合に false を返します。
