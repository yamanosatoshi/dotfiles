Gmagick::drawimage

(PECL gmagick >= Unknown)

Gmagick::drawimage — 現在の画像上の GmagickDraw オブジェクトをレンダリングする

説明

public Gmagick::drawimage(GmagickDraw $GmagickDraw): Gmagick

現在の画像上の GmagickDraw オブジェクトをレンダリングします。

パラメータ

GmagickDraw

画像をレンダリングする描画操作。

返り値

描かれた Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
