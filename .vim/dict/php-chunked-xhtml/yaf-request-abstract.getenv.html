Yaf_Request_Abstract::getEnv

(Yaf >=1.0.0)

Yaf_Request_Abstract::getEnv — ENV 変数を取得する

説明

public Yaf_Request_Abstract::getEnv(string $name, string $default = ?): void

ENV 変数を取得します。

パラメータ

name

変数名。

default

このパラメータを指定すると、変数が見つからない場合にこの値を返すようになります。

返り値

ストリングを返します
