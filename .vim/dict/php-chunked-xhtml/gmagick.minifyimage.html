Gmagick::minifyimage

(PECL gmagick >= Unknown)

Gmagick::minifyimage — 画像をその半分のサイズに比例縮小する

説明

public Gmagick::minifyimage(): Gmagick

画像のサイズを元の半分に比例縮小します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に Gmagick オブジェクトを返します。

エラー / 例外

エラー時に GmagickException をスローします。
