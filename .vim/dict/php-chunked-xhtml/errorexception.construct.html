ErrorException::__construct

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

ErrorException::__construct — 例外を作成する

説明

public ErrorException::__construct(
    string $message = "",
    int $code = 0,
    int $severity = E_ERROR,
    string|null $filename = null,
    int|null $line = null,
    Throwable|null $previous = null
)

例外を作成します。

パラメータ

message

スローする例外メッセージ。

code

例外コード。

severity

例外の深刻度。

注意:

任意の int を設定できますが、ここでは エラー定数 を用いることを想定しています。

filename

例外がスローされたファイル名。

line

例外がスローされた行番号。

previous

ひとつ前の例外。例外の連結に使います。

変更履歴

バージョン 説明
8.0.0 filename と line は、 nullable になりました。 これより前のバージョンでは、 それらのデフォルト値はそれぞれ __FILE__ と __LINE__ でした。
