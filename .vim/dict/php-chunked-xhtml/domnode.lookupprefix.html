DOMNode::lookupPrefix

(PHP 5, PHP 7, PHP 8)

DOMNode::lookupPrefix — 名前空間 URI に基づいて、ノードの名前空間プレフィックスを取得する

説明

public DOMNode::lookupPrefix(string $namespace): string|null

名前空間 URI に基づいて、ノードの名前空間プレフィックスを取得します。

パラメータ

namespace

名前空間 URI。

返り値

名前空間のプレフィックスを返します。
